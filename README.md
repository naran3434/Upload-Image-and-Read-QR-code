# Upload-Image-and-Read-QR-code

# Dependencies 
1. Formidable
2. jimp
3. qrcode-reader

# Installation
1. Clone Repo
2. Install dependency <code>npm install</code>
3. To run <code> node index.js </code>

#API URL
<code> localhost:8700/fileupload </code>

#Method : POST 

#Body params
1. name : filetoupload , type: file
